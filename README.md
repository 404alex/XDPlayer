# XDPlayer


---

主要功能
----
『XDPlayer』是一款多功能的视频播放器。既可以用作本地视频播放器，也可以用来观看IPv6电视，同时支持哔哩哔哩和AcFun的视频播放与下载。您还可以将喜欢的视频收藏起来，方便以后观看。
