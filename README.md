编译环境
----
- Android Studio 2.3.3  
- Gradle 3.3  

主要功能
----
『XDPlayer』是一款多功能的视频播放器。既可以用作本地视频播放器，也可以用来观看IPv6电视。您还可以将喜欢的视频收藏起来，方便以后观看。
